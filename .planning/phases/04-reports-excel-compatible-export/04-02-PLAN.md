<task type="auto">
  <name>Task 1: Create XLSX Export Endpoint</name>
  <files>hr/export.php</files>
  <action>
Create `hr/export.php` — a standalone PHP file that generates and streams a branded multi-sheet XLSX download. This file must NOT include `header.php` or `footer.php` (they produce HTML output which breaks download headers).

**Required includes (in this order):**
```php
require_once __DIR__ . '/../vendor/autoload.php';
require_once __DIR__ . '/../includes/cuti-calculator.php';
require_once __DIR__ . '/../includes/reports-data.php';
```

**Use statements:**
```php
use PhpOffice\PhpSpreadsheet\Spreadsheet;
use PhpOffice\PhpSpreadsheet\Writer\Xlsx;
use PhpOffice\PhpSpreadsheet\Style\Fill;
use PhpOffice\PhpSpreadsheet\Style\Alignment;
use PhpOffice\PhpSpreadsheet\Style\Border;
```

**Empty check:** Get reports via `initReports()`. If `count($reports) === 0`, redirect back: `header('Location: dashboard.php?export_error=empty'); exit;`

**Define color constants** (ARGB format for PhpSpreadsheet):
```php
$COLOR_DEEP_TEAL  = 'FF0F4C5C';
$COLOR_WARM_AMBER = 'FFE8A838';
$COLOR_WHITE      = 'FFFFFFFF';
$COLOR_TEXT       = 'FF1E293B';
$COLOR_BORDER     = 'FFE2E8F0';
```

**Define reusable style arrays:**

`$brandingStyle` — For branding row (A1): Warm Amber text, bold, size 14.

`$timestampStyle` — For timestamp row (A2): Text color secondary, italic, size 10.

`$headerStyle` — For column header rows: Deep Teal fill, white bold text, center alignment, thin bottom border.

`$dataStyle` — For data cells: Text color, thin bottom border, left alignment. Size 11.

---

### Sheet 1: "Ringkasan"

1. Set active sheet title to "Ringkasan"
2. **Row 1 — Branding:** Merge A1:E1, set value "Sicuti HRD — Laporan Cuti Karyawan", apply `$brandingStyle`
3. **Row 2 — Timestamp:** Merge A2:E2, set value "Diekspor: " . date('d/m/Y H:i'), apply `$timestampStyle`
4. **Row 3 — Empty spacer**
5. **Row 4 — Column headers:** A4="No.", B4="Nama Karyawan", C4="Tahun Bergabung", D4="Total Hari Cuti (8 Tahun)", E4="Status Tahun Ini". Apply `$headerStyle` to A4:E4.
6. **Data rows starting at row 5:** For each report (indexed from 0):
   - A = index + 1
   - B = $report['nama']
   - C = $report['tahun_bergabung']
   - D = $report['total_cuti']
   - E = Current year status (call `hitungHakCuti($report['tahun_bergabung'])`, find row where `tahun_kalender == (int)date('Y')`, use its `status`. If no match for current year (e.g., entitlement period doesn't include current year), use "—")
7. **Auto-size columns** A through E: `$sheet->getColumnDimension($col)->setAutoSize(true)`

---

### Sheet 2: "Detail"

1. Create new sheet: `$spreadsheet->createSheet()`, set title "Detail"
2. **Row 1 — Branding:** Merge A1:F1, set value "Sicuti HRD — Detail Entitlement Karyawan", apply `$brandingStyle`
3. **Row 2 — Timestamp:** Merge A2:F2, same timestamp string, apply `$timestampStyle`
4. **Row 3 — Empty spacer**
5. **Row 4 — Column headers:** A4="Nama Karyawan", B4="Tahun Bergabung", C4="Tahun Ke", D4="Tahun Kalender", E4="Hari Cuti", F4="Status". Apply `$headerStyle` to A4:F4.
6. **Data rows starting at row 5:** For each report, call `hitungHakCuti($report['tahun_bergabung'])`. For each of the 8 data rows:
   - A = $report['nama']
   - B = $report['tahun_bergabung']
   - C = $row['tahun_ke']
   - D = $row['tahun_kalender']
   - E = $row['hari_cuti']
   - F = $row['status']
   
   This means each report contributes 8 rows. With 3 seed reports = 24 detail rows.
7. **Auto-size columns** A through F

---

### Set Active Sheet Back to Sheet 1
`$spreadsheet->setActiveSheetIndex(0);`

### Stream Download
```php
header('Content-Type: application/vnd.openxmlformats-officedocument.spreadsheetml.sheet');
header('Content-Disposition: attachment;filename="laporan-cuti-sicuti-hrd.xlsx"');
header('Cache-Control: max-age=0');
header('Pragma: public');

$writer = new Xlsx($spreadsheet);
$writer->save('php://output');
exit;
```

**CRITICAL anti-patterns to avoid:**
- Do NOT include `header.php` or `footer.php` (they output HTML, breaking headers)
- Do NOT use 6-char hex colors — always use 8-char ARGB with FF prefix
- Do NOT write to a temp file — stream directly to `php://output`
- Do NOT forget `exit` after save — prevents any trailing output
